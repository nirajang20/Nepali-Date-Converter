# Nepali Date Converter

A modern, interactive web tool for bidirectional conversion between the English (AD) and Nepali (BS) calendar systems. This application features a sleek, "liquid glass" user interface with dynamic background animations, providing a smooth and engaging user experience for date conversions.

## Features

*   **Bidirectional Conversion:** Seamlessly convert dates from English (AD) to Nepali (BS) and vice-versa.
*   **Interactive Datepickers:** Utilize intuitive date selection controls for both English and Nepali dates.
*   **Day of the Week Display:** Instantly see the corresponding day of the week for selected dates in both English and Nepali.
*   **Modern UI/UX:** Enjoy an ultra-modern design with liquid glass effects, animated aurora backgrounds, floating SVG blobs, and subtle particle animations for an aesthetically pleasing interface.
*   **Responsive Design:** Optimized for various screen sizes, from desktops to mobile devices.

## Technologies Used

*   **HTML5:** Structure of the web page.
*   **CSS3:** Styling, including advanced animations, `backdrop-filter`, and gradients for the unique "liquid glass" aesthetic.
*   **JavaScript:** Core logic for date conversion and UI interactions.
*   **Nepali Datepicker v5.0.6:** A robust JavaScript library by Sajan Maharjan for handling Nepali calendar logic and date input.
    *   [Official Website](https://nepalidatepicker.sajanmaharjan.com.np/v5/)
*   **jQuery:** A fast, small, and feature-rich JavaScript library for DOM manipulation and event handling.
*   **jQuery UI:** Provides the English datepicker widget and enhances UI interactions.

## Usage

1.  **Open `index.html`:** Simply open the `index.html` file in your web browser.
2.  **Select a Date:**
    *   Click on the "English Date (AD)" input field to open the English datepicker and select a date. The Nepali equivalent will automatically appear in the "Nepali Date (BS)" field.
    *   Click on the "Nepali Date (BS)" input field to open the Nepali datepicker and select a date. The English equivalent will automatically appear in the "English Date (AD)" field.
3.  **View Day of the Week:** The current day of the week for the selected date will be displayed below the converter in both English and Nepali.

## Setup / Installation

No complex installation is required.
1.  Clone this repository or download the project files.
2.  Navigate to the project directory.
3.  Open `index.html` in your preferred web browser.
